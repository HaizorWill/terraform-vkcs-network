# VKCS Network Terraform Module

Terraform module that creates a network with subnets in VK Cloud.

It aims to provide a flexible and simplified way to create networks in VK Public Cloud.

> It may also work with VK Private Cloud, as it is not tightly coupled to Public Cloud only. However, it has not been tested with Private Cloud and it is not planned to be in a foreseable future.

## Features

It supports creating:

 * A single network
 * Multiple subnets in the network
